## 🚀 Configuración del entorno de desarrollo con React

Este proyecto utiliza **React + TypeScript + Vite** y requiere **Node.js** para ejecutarse.

### Requisitos previos
- Instalar [Node.js](https://nodejs.org/) (versión 18 o superior).
- npm (se incluye automáticamente con Node.js).
- Un editor de código como [Visual Studio Code](https://code.visualstudio.com/).

### Pasos para instalar y ejecutar
Abrir tu terminal (CMD o la terminal de VS Code).Es el punto de partida.
1. Clonar el repositorio
git clone https://github.com/Top-Music/Reserva-de-Restaurante.git

2. Entrar en la carpeta del proyecto usando cd:
cd Reserva-de-Restaurante

3. Instalar las dependencias:
npm install

4. Ejecutar el proyecto en modo desarrollo:
npm run dev

5. Abrir el navegador en la dirección que te muestre la terminal (normalmente http://localhost:5173).
